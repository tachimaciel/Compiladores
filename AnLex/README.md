Analizador Léxico para archivos JSON hecho en el lenguaje C

# Windows
Compilación y Ejecución:
  * gcc anlex.c tablaSimbolos.c -o salida
  * salida.exe fuente.txt
  
# Linux
Compilación y Ejecución:
  * gcc anlex.c tablaSimbolos.c -o salida
  * ./anlex fuente.txt
